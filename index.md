# 작심삼일?  
# 해제는 유료.

해야지 말고, 하자.

**작심유료**는 알고도 실천하지 못하는 사람들을 위한 집중 잠금 앱입니다.  
유튜브, 배달앱, SNS, 브라우저 등 내가 직접 고른 앱을 집중 시간 동안 잠그고, 해야 할 일에 몰입할 수 있게 도와줍니다.

앱은 무료로 사용할 수 있습니다.  
다만 집중 시간 중 꼭 잠금을 풀어야 한다면, 사용자가 직접 잠금 해제 기능을 구매해 해당 세션을 해제할 수 있습니다.

---

## 주요 기능

### 집중시간 동안 앱 차단

사용자가 직접 선택한 앱을 집중 시간 동안 차단합니다.

### 중간 해제는 본인 선택

정말 필요한 경우에만 사용자가 직접 잠금 해제 기능을 구매해 세션을 해제할 수 있습니다.

### 집중 기록

완료한 집중 세션과 중간 해제 기록을 확인할 수 있습니다.

### 주간 작심 리워드

잠금 해제 기능 구매 금액의 일부는 주간 작심 리워드 운영에 사용될 수 있습니다.  
리워드는 이벤트성으로 운영되며, 지급 기준과 내용은 앱 내 공지에 따릅니다.

---

## 개인정보 보호

작심유료는 키 입력, 메시지, 화면 내용, 연락처, 위치 정보를 수집하지 않습니다.

접근성 권한은 사용자가 선택한 앱 실행 감지와 차단 화면 표시 목적으로만 사용됩니다.

자세한 내용은 [개인정보처리방침](./privacy.html)을 확인해 주세요.
